# ⛽ Petrol Tracker - Fleet Management System

A comprehensive web-based fleet management system for tracking fuel consumption, vehicle usage, and generating analytics for government or corporate fleets.

## ✨ Features

- **Vehicle Management**: Register, edit, and delete vehicles with fuel type (Petrol/Diesel)
- **Fuel Tracking**: Record fuel purchases with automatic consumption calculation (L/100km)
- **Monthly Reports**: View usage records by month with totals and averages
- **Driver Tracking**: Flag when vehicle is used by non-custodian drivers with remarks
- **Analytics Dashboard**: Visual reports on fuel consumption, costs, and trends
- **OCR Support**: Scan fuel receipts for automatic data entry (coming soon)
- **Responsive Design**: Works on desktop, tablet, and mobile devices
